# Допомога Поруч: Новояворівськ 🇺🇦

## Про проект
"Допомога Поруч" — це інформаційний волонтерський веб-хаб. Проект розроблено в рамках розрахунково-графічної роботи з курсу "Управління ІТ-проектами". 
Мета платформи — консолідація актуальних волонтерських потреб, зборів та ініціатив (плетіння сіток, допомога ЗСУ, підтримка притулків) в єдиному місці.

## Технології
- HTML5
- CSS3
- JavaScript (ES6)

## Структура
- `index.html` — базовий каркас сторінки.
- `style.css` — стилізація та адаптивність інтерфейсу.
- `script.js` — логіка генерації карток (імітація отримання даних з бази).
